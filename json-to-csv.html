FAQ

Is my JSON data uploaded anywhere?

No. The conversion happens entirely in your browser with JavaScript. Your data is never sent to a server, so it's safe for API exports, customer records, and other sensitive data.

What JSON shapes are supported?

An array of objects (keys become the header row — keys are merged across all objects), an array of arrays (each inner array becomes a row), or a single object. Nested objects and arrays are embedded as JSON strings in their cell.

Will the CSV open in Excel or Google Sheets?

Yes. Fields containing commas, quotes, or line breaks are quoted per the CSV standard (RFC 4180), so the file imports cleanly. If your locale's Excel expects semicolons, switch the delimiter option.
